# Weather Forecast Dashboard

A simple React-based weather forecast dashboard, using <a href="openweathermap.org">Open Weather Map</a>'s 5 day / 3 hour API to show the weather conditions or plot the temperatures

# Getting Started

1. Clone the project to your machine by running the following command:

    `git clone https://github.com/Alireza3044/weather-forecast-dashboard.git`

2. Create an account on <a href="openweathermap.org">Open Weather Map</a>.

3. Get your API key and then create a `.env` file in the projects root directory and provide your API key to `VITE_API_KEY` variable.

4. Run `npm i` command to install the required packages.

5. Now run the server by running the command `npm run dev`.
